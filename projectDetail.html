SPORTS SIMULATOR

Navigate to Github for Project Code

Python & Django

Description

This is a College Football Simulation. Schedule generation, 1 of 16 available teams, team abilities directly computated from team's current player's abilities, and team abilities directly influence their odds of win each game are just a few of the features. Eventually, I will build in the ability for the user to further control their team's destiny by recruiting incoming players. Recruits will further be influenced by factors such as championships, historical win percentages, and recruiting points allocated to the specific recruit by the user.

Highlights
  • Extensive Algorithms: schedule generation, weighted game simulations, year-by-year player growth & rotation
  • Year-by-year player growth & rotation: After each year, all 4-year players & potential 3-year players, every players's year is increaced and their ability is increased at random to simulate player growth
  • Stats & Standings: Both current and historic stats are recorded and displayed as the game progresses from weeks to seasons.

SHARE A QUOTE

Navigate to Github for Project Code

Python & Django

Description

This is a friendly application that gives the users a corner of the world to pay tribute to their most precious words. Users have the ability to create a quote by entering the name of the person responsible for the words and the message itself. The quotes could be posted because they are powerful, inspiring or just funny. The quotes posted will give tribute to both the person who first spoke or wrote those words, but also the user who decided to post them. If the user potentially made an error, then they can go back in and edit the quote or delete it all together. Now that the users can review quotes posted by others and add quotes they like to their favorites and if they change their mind can remove them.

Highlights
  • Login and registration built to shutout all users unless properly authorized (process tested by instructors). Only quotes created by the current user can be effected (edit, deleted, favorited or unfavorited)
  • Users can view everyone's posted quotes, add quotes to their favorited section, and later on they can be removed from their favorites section
  • Users can review the other users profiles as well. By clicking any user's name, you can see all quotes posted by the user as well as the count of quotes

PET ADOPTION AGENCY

Navigate to Github for Project Code

MEAN

Description

From the comfort of your home, using the PET ADOPTION AGENCY, you can review the available pets, like them, adopt them, add new ones, or edit he pets you have added. To add them, Let us know what type of pet, a brief description, important skills they possess, and we'll do the rest. We accept all animal type from Zebras to cobras.

Highlights
  • Using Angular, front-end validations have been applied to the edit/new form through disabling the submit button and displaying errors below the input fields to prevent users from entering invalid information.
  • Using mongoose, back-end validations have been applied to further secure that accurate information has been entered.
  • If a user chooses to adopt the pet, the pet is then removed from the website.
  • Back-end logic has been set in place to prevent a user from adding a pet with the same name as another pet in the system. Any attempt to add a pet with the same name will result in an error message.

Description
