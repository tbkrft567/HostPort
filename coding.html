Coding Dojo is a technical bootcamp that teaches students to code in 3 different stacks over a 14 week period, Python, Ruby on Rails, and MEAN. At the end of each stack, students are challenged to complete a fully functioning application in under 5 hours. If a accomplished, then the student recieves a red belt, but if a student can manage the advanced challenges in the same timeframe, then they are awarded the black belt.
